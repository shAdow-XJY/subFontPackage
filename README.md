# subFontPackage 字体包压缩

字体包体积大小按需压缩。
根据选择的目录或者文件，读取所需的文字，对原本的字体包进行剔除多余的内容，从而生成轻量的字体包。

## 支持环境

1. 支持Windows
2. macos和linux可以实现，也不难，但是还没做，需要我有一个macos和linux的电脑跑一下生成。
